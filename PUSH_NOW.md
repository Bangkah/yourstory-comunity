## ‚ùì KALAU ADA MASALAH

### Git belum installed?
```bash
sudo apt-get install git
```

### Git credentials belum setup?
```bash
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
```

### Lupa GitHub password/token?
GitHub sekarang butuh Personal Access Token bukan password. Buat di:
https://github.com/settings/tokens

Pilih: `repo` + `workflow` permissions

### Remote sudah ada?
```bash
git remote remove origin
git remote add origin https://github.com/Bangkah/yourstory-comunity.git
```

### Sudah push tapi lupa tag?
```bash
git tag -a v1.0.0 -m "Version 1.0.0: Initial release"
git push origin v1.0.0
```
